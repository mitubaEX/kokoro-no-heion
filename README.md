# 心の平穏 — Kokoro no Heion

ヒーリングミュージックと先人の教え、瞑想ガイドを通じて、心の平穏を促す Web アプリです。
すべての音は **Web Audio API でその場で生成**しており、音源ファイルを使わないため著作権の心配がありません。データはすべて端末内に保存され、外部に送信されません。

## 主な機能

- 🎵 **ヒーリングミュージック** — 波・雨・森・焚き火・虫の音・ホワイトノイズ・シンギングボウル・星空のパッドなど **8 種**のサウンドスケープ。音量調整・就寝タイマー（フェードアウト停止）付き。
- 🧘 **瞑想ガイド** — 呼吸アニメーションと音のトーン誘導。4-7-8・ボックス・5-5・ため息・朝の活性など **5 種**の呼吸パターン、セッション長の選択。
- 📜 **先人の教え** — 出典の確かな、パブリックドメインの言葉を厳選。テーマ別に閲覧できます。
- 📔 **心の日記** — その日の気分（5 段階）とひとことメモを記録・振り返り（端末ローカル保存）。
- 🏠 **ホーム** — 時刻別あいさつ、今日の一言、利用履歴に基づくおすすめ。
- 💬 その他 — アプリ内フィードバック、PWA（オフライン対応・ホーム画面に追加可能）、アクセシビリティ配慮（キーボード操作・`prefers-reduced-motion`）。

## 技術スタック

- React 18 + TypeScript + Vite
- React Router (HashRouter)
- Web Audio API（手続き的サウンド生成）
- vite-plugin-pwa（Service Worker / manifest）
- Vitest（ユニットテスト）

## 開発

```bash
npm install      # 依存をインストール
npm run dev      # 開発サーバ → http://localhost:5173/
npm run build    # 本番ビルド（型チェック + Vite + PWA）
npm run preview  # ビルド結果をプレビュー
npm test         # テスト実行
```

## プロジェクト構成

```
src/
  audio/      Web Audio エンジンとサウンドスケープ生成
  components/ AudioGate（音声有効化）・FeedbackSheet・アイコン
  data/       教え・呼吸パターン・今日の一言
  lib/        永続化・設定・日記・利用履歴・フィードバック
  pages/      ホーム / 音 / 瞑想 / 教え / 日記
```

開発の経緯・残課題は [`ISSUES.md`](./ISSUES.md)、依存の脆弱性方針は [`SECURITY.md`](./SECURITY.md) を参照してください。
